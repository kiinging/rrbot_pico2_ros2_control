◆ loop()

void PCA9685_servo::loop ( uint64_t  TEllapsed)

Routine to call every loop iteration.

Note
Enclose this in a for()/while() loop and call every time, passing TEllapsed argument containing the time passed since loop() was last called.
E.g.
TNow = time_us_64(); // time now in microseconds
TEllapsed = TNow - TPrevious; // time, in microseconds, since the last loop
TPrevious = TNow; // store this ready for the next loop
servo.loop(TEllapsed);
Parameters
TEllapsedtime since last function call in microseconds
